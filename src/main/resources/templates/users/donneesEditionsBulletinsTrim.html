Paramètres de production des bulletins trimestriels
Sélectionner la classe et le trimestre puis cliquer sur le bouton envoyer pour que les bullletins trimestriels soient édités. Une fois cela fait, le fichier les contenant vous sera proposé en téléchargement.
NB:
Une fois les bulletins générés appuyer sur la touche F5 ou actualiser la page pour générer le rapport de conseil de classe.
Si une erreur concernant la section survient, faites vérifier par l'administrateur si les codes des sections sont soit: GENERAL, INDUSTRIELLE OU COMMERCIALE
